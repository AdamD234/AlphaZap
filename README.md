# Word Zap Game

A browser-based word puzzle game where players create words to eliminate letters from a challenge word. The goal is to strategically form words so that matching letters cancel out until all letters are removed.

## How to Play

1. A **random word** is selected as the challenge word.
2. Create words using the available letters.
3. When a **pair of the same letter appears, they get zapped (removed)**.
4. Continue forming words to remove all letters.
5. **Zap all the letters to win**.

After completing a challenge word, a new random word will be selected, or you can choose your own.

## Features

- Random challenge word generation
- Add your own words to the current letters
- Undo your last turn
- Restart with the current words
- Restart with a new challenge word
- Choose your own starting word (letters only, no spaces)
- Alphabetize the current list of letters
- Hover over words to see their definitions
- Click word definitions to learn more about the word
